# Module List

Sistem SEHATI dibagi menjadi beberapa modul independen. Setiap modul akan memiliki struktur internal (Controller, Service, Model) yang terisolasi.

## 1. Authentication Module
- Login, Logout
- Role Management (Super Admin, Admin Posyandu)
- Reset Password

## 2. Public Module
- Landing Page
- Edukasi & Panduan

## 3. Screening Module (Core)
- Kuesioner Interaktif (20 Variabel)
- Rule Engine v2 (GATEC) Evaluator
- Result Generator (Scoring & Classification)

## 4. Respondent Module
- Manajemen Data Demografi Responden
- Riwayat Skrining Responden

## 5. Master Data Module
- Manajemen Desa
- Manajemen Posyandu
- Faskes Rujukan

## 6. Dashboard & Analytics Module
- Overview 5 PTM (Diri Sendiri & Keluarga)
- Filter by Desa/Kecamatan
- Export Data (Excel/PDF)
